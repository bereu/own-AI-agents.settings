# overview 
* define spec with grill-me skill with communicator 
* create spec. markdown document 
* pass to execute with executor agent
* execute agent make changes with implement.md document.
* do code review with validator agent
* validate the changes and testing behaviour and update test.md of checklist. 
* if it failed Please feedback to executor-agent and re-work.
* update implement.md and test.md and committ the code changes.
